### Background
This is a responsive blog web app done after finishing a portion of the [Web Dev Bootcamp](https://www.udemy.com/course/the-complete-web-development-bootcamp/?couponCode=ST2MT110724BNEW) in Udemy

### Step 1
Run `npm install`

### Step 2
Run `npm run dev`
